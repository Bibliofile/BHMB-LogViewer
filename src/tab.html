Tips:

  • You can use online=NAME to show chat from when someone is online
  • You can use sender=NAME to show chat from NAME
  • You can use context=# to show # messages around matched messages
  • To use names with spaces, remove the space.
  • Search is case-insensitive.
  • online=NAME1 online=NAME2 will show chat when both NAME1 and NAME2 are online.
  • sender=NAME1 sender=NAME2 will show chat from both NAME1 and NAME2.
