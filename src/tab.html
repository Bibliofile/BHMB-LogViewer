Options





  • Search is case-insensitive.
  • Use online=NAME to show chat when NAME is online.
  • Use online=NAME1 online=NAME2 to show chat when NAME1 and NAME2 are both online.
  • Use sender=NAME to show chat from NAME.
  • Use context=x to show x number of messages around matched messages.
  • Example: Use context=5 sender=bob to show five messages before and after messages from Bob.
  • To use names with spaces, remove the space.
  • Example: Use sender=bobt to show chat from BOB T.
